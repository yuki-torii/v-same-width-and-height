## Install
```bash
$ npm i --save v-same-width-and-height
# or
$ yarn add v-same-width-and-height
```

## Usage
```js
import Vue from 'vue'
import vSameWidthAndHeight from 'v-same-width-and-height'

vue.use(vSameWidthAndHeight)
```

```html
<img v-same-width-and-height src="foo.png" alt="foo">
```
